                (readonly) chats :Object

                Source:
                • chat-engine/src/index.js, line 831

                An object containing the Chats this User is currently in. The key of each item in the object is the Chat.channel and the value is the Chat object. Note that for privacy, this map will only contain Chats that the client (Me) is also connected to.

                Type:
                • Object
                Example
                {
   "globalChannel": {
       channel: "globalChannel",
       users: {
           //...
       },
   },
   // ...
}

                direct :Chat

                Source:
                • chat-engine/src/index.js, line 849

                Direct is a private channel that anybody can publish to but only the user can subscribe to. Great for pushing notifications or inviting to other chats. (security not implemented yet).

                Type:
                • Chat

                feed :Chat

                Source:
                • chat-engine/src/index.js, line 840

                Feed is a Chat that only streams things a User does, like 'startTyping' or 'idle' events for example. Anybody can subscribe to a User's feed, but only the User can publish to it (security not yet implemented).

                Type:
                • Chat

                (readonly) uuid :String

                Source:
                • chat-engine/src/index.js, line 805

                The User's unique identifier, usually a device uuid. This helps ChatEngine identify the user between events. This is public id exposed to the network. Check out the wikipedia page on UUIDs.
